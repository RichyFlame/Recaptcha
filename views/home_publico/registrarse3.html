Configuración de cuentas


Selecciona los permisos que cada usuario realizará y agrega los números de cuenta que deseas asociar.

Consulta: El usuario que tenga asignado este permiso únicamente podrá realizar consulta de saldo y Estados de Cuenta.

Pagos y transferencias: Con este permiso el usuario podrá realizar transferencias entre cuentas propias, pagos de Préstamo y Tarjetas de Crédito.

Bi Agencia: Quien tenga este permiso podrá realizar solicitudes de chequeras, bloqueo y desbloqueo de cuentas y cheques.

Banca SAT: Esta opción te permite realizar pagos de impuestos, Declaraguate y pólizas de importación. Para tener acceso a esta opción debes generar y adjuntar el contrato de Banca SAT.
