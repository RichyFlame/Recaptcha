¿Cómo funciona?

Al igual que el token físico, esta aplicación te brindará códigos aleatorios para poder autenticarte en tu usuario Bi Banking.


1. Descarga la aplicación HID Approve desde Google Play o App Store.

2. Abre la aplicación y escanea el código QR que aparecerá en tu página de Bi Banking.

3. ¡Listo! La aplicación será configurada exitosamente con tu usuario de Bi Banking.
